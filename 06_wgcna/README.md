# 06 — WGCNA

Weighted Gene Co-expression Network Analysis. Builds feature × feature networks within a block, identifies modules of co-varying features, and integrates across blocks via eigengene correlations.

## What is in this folder

- [`wgcna.qmd`](wgcna.qmd) — rendered as the *Tutorial 5: WGCNA* page of the site.

## Library used

[`WGCNA`](https://horvath.genetics.ucla.edu/html/CoexpressionNetwork/Rpackages/WGCNA/) — Langfelder P, Horvath S. *BMC Bioinformatics* 9:559 (2008).
